System Block Diagram - Binary Arithmetic Digital Calculator

Keyboard Input
User enters binary numbers and selects an operation (addition, subtraction, multiplication, or division).
Arithmetic Logic Unit (ALU)
The ALU processes binary operations using the adder, subtractor, multiplier, and divider circuits.
Binary Adder
Performs addition of two binary numbers using logic gates.
Binary Subtractor
Handles binary subtraction using two's complement representation.
Binary Multiplier
Executes binary multiplication using the shift-and-add method.
Binary Divider
Carries out binary division using restoring or non-restoring division methods.
Output Display
Shows the computed results in binary and decimal format for user interpretation.
